module grilleFrame

grilleFrame est un module utilisé uniquement par la version graphique du jeu. Ce module contient l’implémentation de la classe GrilleFrame qui est très importante pour la version graphique du projet, car il s’agit en quelque sorte du terrain sur lequel se déroule le jeu de la vie.

class grilleFrame.GrilleFrame(master, nbLignes: int, nbColonnes: int, typeg: str)[source]

Bases : tkinter.Frame, grille.Grille

Cette classe hérite de la classe Grille et permet d’implémenter la grille sur laquelle se déroule le jeu de la vie qui peut être plane ou torique. master: il s’agit du composant graphique parent de la grille. nbLignes: nombre de lignes de la grille. nbColonnes: nombre de colonnes de la grille. typeg: type de grille (plane ou torique).

Méthode qui permet de changer l’état d’une cellule de la grille lorsqu’on clique dessus.

on_cell_hover(event)[source]

Méthode qui permet de changer la couleur de fond d’une cellule de la grille lorsqu’elle est survolée.

static on_cell_leave(event)[source]

Méthode qui permet de rétablir la couleur de fond d’une cellule lorsqu’elle n’est plus survolée.

remplir_alea(taux=10)[source]

Méthode qui permet de remplir aléatoirement la Grille avec un certain taux de cellules vivantes. Le paramètre « taux » représente le pourcentage de cellules vivantes à créer sur la grille.
