module app

app est le module principal de la version graphique du projet. C’est dans ce module que la fenêtre principale du jeu est créée à l’aide de librairie tkinter. Il s’agit en effet du point de départ du projet.

class app.App(minWidth: int, minHeight: int, title: str)[source]

Bases : tkinter.Tk

Classe principale du projet qui se charge de créer les différents composants graphiques. C’est également la fenêtre principale du jeu. minWidth: largeur minimum en pixels de la fenêtre. minHeight: hauteur minimum en pixels de la fenêtre.

NBCOLONNES = 50

attribut de classe qui indique le nombre de colonnes de la grille.

NBLIGNES = 30

attribut de classe qui indique le nombre de lignes de la grille.

alea_init_config()[source]

Méthode qui permet d’initialiser de façon aléatoire la grille avec un taux de cellules vivantes fourni par l’utilisateur. Cette méthode est appelée après un click de l’utilisateur sur le bouton Initialiser.

get_alea_rate()[source]

Méthode qui permet de récupérer la proportion initiale de cellules vivantes entrée par l’utilisateur.

init_known_struct()[source]

Méthode qui permet d’initialiser la grille avec la configuration initiale d’une structure connue (Clignotant, Ruche 1, Ruche 2, Balise, Crapeau, Escalier, Pulsar, Pentadecathlon ou Vaisseau spatial)

play_niter()[source]

Méthode qui utilise les règles du jeu de la vie pour mettre la grille à jour de façon itérative pendant un nombre d’itérations fourni par l’utilisateur

reset_grid()[source]

Méthode qui permet de réinitialiser la grille, c’est-à-dire la remettre à son état initial avec toutes les cellules mortes. Cette méthode est appelée après un click de l’utilisateur sur le bouton Réinitialiser.

update_grid_type(event)[source]

Méthode qui permet de changer le type de grille (plane ou torique) à utiliser et de recalculer les voisins de chaque cellule en conséquence

static validate_numeric_entry(user_input)[source]

Méthode de classe qui permet d’effectuer la validation d’un champ de texte numérique, c’est-à-dire faire en sorte que ce champ de texte n’accepte que des valeurs numériques
