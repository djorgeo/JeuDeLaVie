module bibli_exemples

bibli_exemples (bibliothèque d’exemples) est un module dans lequel quelques exemples de structures connues ont été implémenté. Ce module permet en outre de mettre à la disposition de l’utilisateur des exemples lui permettant de comprendre quel est l’utilité du jeu de la vie.

bibli_exemples.balise(self)[source]

Initialise la grille avec la configuration initiale qui permet de simuler une balise.

bibli_exemples.crapeau(self)[source]

Initialise la grille avec la configuration initiale qui permet de simuler un crapeau.

bibli_exemples.escalier(self)[source]

Initialise la grille avec la configuration initiale qui permet de simuler un escalier qui se transforme en 4 blocs stables.

bibli_exemples.pentadecathlon(self)[source]

Initialise la grille avec la configuration initiale qui permet de simuler un pentadecathlon qui oscille avec une période de 15.

bibli_exemples.pulsar(self)[source]

Initialise la grille avec la configuration initiale qui permet de simuler un pulsar qui oscille avec une période de 3.

bibli_exemples.ruche1(self)[source]

bibli_exemples.ruche2(self)[source]
