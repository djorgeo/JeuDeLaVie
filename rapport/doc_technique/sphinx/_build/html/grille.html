module grille

grille est un module utilisé à la fois par la version graphique et non graphique du jeu. Ce module contient l’implémentation de la classe Grille qui est l’une des plus des plus importantes du projet.

class grille.Grille(nbLignes: int, nbColonnes: int, typeg: str)[source]

Bases : object

Cette classe permet d’implémenter la grille sur laquelle se déroule le jeu de la vie qui peut être plane ou torique. nbLignes: il s’agit du nombre de lignes de la grille. nbColonnes: représente le nombre de colonnes de la grille. typeg: c’est le type de grille (plane ou torique).
